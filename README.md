# Food Website

This full stack app is meant to be utilised in two ways:

1. A user can choose what kind of food s/he likes and place an order.
2. An admin can alter the menu, the quantity, etc.

# Tech stack:

Frontend --> HTML, CSS, Javascript
Backend --> PHP
Database --> MySQL
